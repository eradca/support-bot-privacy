# Privacy Policy for Support Bot

**Last Updated:** November 8, 2025

Support Bot does not collect, store, or share any personal data from users.

The bot only processes the following information temporarily in order to function properly:
- Discord user ID
- Server ID
- Channel ID
- Message content (only when executing commands)

No data is permanently stored, and no information is shared with any third parties.

If you believe your data has been stored or you wish to have it removed, please contact the bot owner.

**Contact:** supportbotteam@gmail.com or via Discord (Support Bot Owner)
